# LoanLite – Android Loan Management App

LoanLite is an Android application designed to manage loan applications, user verification, and loan tracking workflows with a Firebase backend.

## App Preview

[Screenshots](https://github.com/Code-Strap/LoanLite/tree/e5c4fef19e0dfaaf634dc68e5ca8d0215113ccc5/Screenshots)

## Features
- User authentication using Firebase Authentication
- Loan application and status tracking
- Document upload and verification flow
- Separate user and admin dashboards
- Payment screen for repayments
- Conditional UI based on verification and loan status

## Tech Stack
- Java
- XML
- Android SDK
- Firebase Authentication
- Firebase Firestore

## Project Status
In progress – core features implemented, further improvements planned.

## Purpose
Personal learning project focused on Android development, backend integration, and real-world application workflows.
